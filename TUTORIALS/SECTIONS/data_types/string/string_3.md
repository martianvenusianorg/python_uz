### .strip() metodi

```python
str = ' Python '
print("'"+str+"'")
# ' Python '

strip_str = str.strip()
print("'"+strip_str+"'")
# 'Python'

print(type(strip_str))
# <class 'str'>
```

### .lstrip() metodi

```python
str = ' Python '
print("'"+str+"'")
# ' Python '

strip_str = str.lstrip()
print("'"+strip_str+"'")
# 'Python '

print(type(strip_str))
# <class 'str'>
```
