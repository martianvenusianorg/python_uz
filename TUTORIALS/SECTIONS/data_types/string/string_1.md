### title() metodi.
```python
name = 'ism familiya'
print(name.title())
# Ism Familiya
```

### upper() metodi.
```python
name = 'bugung tong otdi. Va yana quyosh botdi!'
name_upper = name.upper()
print(name_upper)
# BUGUNG TONG OTDI. VA YANA QUYOSH BOTDI!
```

### lower() metodi.
```python
name = 'BUGUNG TONG OTDI. VA YANA QUYOSH BOTDI!'
name_lower = name.lower()
print(name_lower)
# bugung tong otdi. va yana quyosh botdi!
```

### capitalize() metodi.
```python
name = 'bugung tong otdi. va yana quyosh botdi!'
print(name.capitalize())
# Bugung tong otdi. va yana quyosh botdi!
```
