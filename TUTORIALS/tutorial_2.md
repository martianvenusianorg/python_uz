# Boolean turlar

`True` va `False` (rost va yolg'on) turlar

```python
print(True)
# True
```

```python
print(False)
# False
```

```python
condition = True
print(condition)
# True
```

```python
condition = False
print(condition)
# False
```

Booleanning turini tekshirib ko'ramiz

```python
condition = True
condition_type = type(condition)
print(condition_type)
# <class 'bool'>
```
